# Sakshi Shinde - Data Analyst Portfolio

This is a modern, high-performance portfolio built with TanStack Start, React 19, and Tailwind CSS.

## Features

- **TanStack Start**: Full-stack React framework with SSR and streaming.
- **3D Visuals**: Integrated React Three Fiber for immersive data storytelling.
- **Responsive Design**: Mobile-first approach using Tailwind CSS.
- **SEO Optimized**: Built-in meta tag management and semantic HTML.

## Tech Stack

- **Framework**: TanStack Start / React 19
- **Styling**: Tailwind CSS 4.0
- **Animations**: Framer Motion
- **3D**: Three.js / React Three Fiber
- **Icons**: Lucide React
- **Routing**: TanStack Router

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Deployment

This project is configured for easy deployment on **Vercel**.
The build output is automatically directed to the `.output` directory.
